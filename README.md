# Lumira AI

Lumira AI is a next-generation personal fashion assistant that helps you discover, style, and shop for outfits using smart AI features. It combines virtual try-on, skin analysis, voice commands, and a connected shopping experience — all in one seamless platform.

## Features

- **Virtual Try-On** — Upload a photo and see how outfits look on you
- **Skin Analysis** — AI-powered skin health insights
- **Voice Commands** — Control the app hands-free with your voice
- **Smart Shopping** — Browse and shop curated fashion catalogs
- **Daily Dashboard** — Personalized outfit and style recommendations
- **Health Vitals** — Track your wellness alongside your style
- **Pi Network Wallet** — Integrated payment support

## Tech Stack

- ⚛️ **React 19** with TanStack Start
- 🎨 **Tailwind CSS** for styling
- 🔒 **Supabase** for backend & authentication
- 🤖 **Gemini AI** for intelligence features
- ☁️ **Cloudflare Pages** for deployment

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/alhdada620-rgb/lumira-ai.git
   cd lumira-ai
   ```

2. Install dependencies:
   ```bash
   bun install
   ```

3. Set up your `.env` file with your credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_key
   ```

4. Start the dev server:
   ```bash
   bun run dev
   ```
